Projects

Adobe Systems

Noida, India
May 2016 to July 2016

    My project was to implement a search query autocompletion feature for a website hosted on a static server. I used NLP techniques to build the feature. The main challenge was to make faster predictions by minimizing the size of data downloaded from the server. We designed and implemented a structure to distribute the model into various files based on words. The client side application was designed to download a minimal set of files to make predictions for a given query. Heuristics like predicting a word till the required files are downloaded improved the user-experience further.
